# 📈 AdSales Prediction AI

An enterprise-grade, AI-powered web application that predicts product sales based on advertising expenditure across different marketing platforms such as TV, Radio, and Newspaper ads.

## 📌 Project Overview

AdSales Prediction AI helps businesses and marketers estimate future sales performance using Machine Learning algorithms. Users can input advertising budgets and receive predicted sales results instantly. The application has been modernized into a full-stack Next.js web application with a professional, dark-themed dashboard, secure authentication, and a responsive user interface.

The project demonstrates the practical implementation of:
- Full-Stack Web Development (Next.js & React)
- Secure User Authentication (NextAuth & Google OAuth)
- Modern UI/UX Design (Tailwind CSS)
- AI-based Prediction Systems Integration
- Interactive Web Deployment (Vercel)

## ✨ Key Features

## ⚙️ Installation & Setup

Follow these steps to run the project locally on your machine.

### 1. Clone the Repository

```bash
git clone https://github.com/Aneela123-Lab/adsales-prediction-ai.git
cd adsales-prediction-ai
```

### 2. Install Dependencies

You can use `npm`, `yarn`, or `pnpm`:

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env.local` or `.env` file in the root directory and add the following keys for authentication to work:

```env
NEXTAUTH_SECRET=your_super_secret_random_string
NEXTAUTH_URL=http://localhost:3000
GOOGLE_CLIENT_ID=your_google_cloud_client_id
GOOGLE_CLIENT_SECRET=your_google_cloud_client_secret
```

### 4. Run the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
